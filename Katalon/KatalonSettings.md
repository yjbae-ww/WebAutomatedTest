## Katalon 설치

### 선수 설치
- JDK 설치
- Android Studio 설치
- Node.js 설치

### 환경 변수 설정
- JAVA_HOME
- ANDROID_HOME

### Appium 설치
1. 설치
```
npm install -g appium
```
2. web driver 설치
```
npm install wd
```
3. version 체크
```
appium –version
```
4. docotr 설치 (선택) /appium 보조 기능
```
npm install -g appium-doctor
```
5. uiautomator2 설치 
```
appium driver install uiautomator2
```
